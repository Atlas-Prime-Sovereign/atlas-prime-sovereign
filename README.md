## ğŸš¦ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Google Gemini API Key ([Get one here](https://ai.google.dev/))

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Atlas-Prime-Sovereign/atlas-prime-sovereign.git
   cd atlas-prime-sovereign
   ```

2. **Open index.html**
   - Simply open `index.html` in your browser
   - No build process required!

3. **Configure API Key**
   - On first launch, you'll be prompted to enter your Gemini API key
   - The key is stored securely in your browser's local storage

### Usage

1. **Select a Governance Mode** from the sidebar
2. **Enter your query** in the chat interface
3. **Use voice input** by clicking the microphone icon
4. **Upload documents** for AI analysis
5. **Access specialized tools** via the Quantum Hub
